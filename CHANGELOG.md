# Changelog

## [0.1.0] - 2025-03-11

### Added

- Initial release of UIX Framework
- XML parser for component and screen templates
- USS (CSS-like) style parser
- Runtime rendering to uGUI
- Reactive binding system
- Navigation system
- Theme support with hot-swap
- Asset pipeline (compile, validate, prefab generation)
- UACF integration (19 tools)
- Editor tools (Preview, Scaffolders, Inspectors)
- Built-in components (Button, Text, Card, etc.)
- Samples and tests
